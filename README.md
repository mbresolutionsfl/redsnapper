# Red Snapper Realty - Production-ready Next.js Site

This is a minimal production-ready Next.js (app router) project scaffold containing your Red Snapper Realty site.

## How to use

1. Unzip the project and open a terminal in the project folder.
2. Install dependencies:
   ```
   npm install
   ```
3. Run development server:
   ```
   npm run dev
   ```
4. Build for production:
   ```
   npm run build
   ```
5. Deploy to Vercel: connect this repository to Vercel â€” it will auto-detect Next.js.

## Notes
- Tailwind is configured; if you prefer plain CSS you can remove Tailwind files.
- Add any images to the `public/` folder and reference them using `/your-image.png`.
